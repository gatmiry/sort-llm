How to reproduce the plots in this folder
==========================================

Both plots compare models trained with and without layer normalization
on the sorting task (vocab_size=128, block_size=32, 1 head, 2 layers).

1. count_comparison.png  (Left panel: candidate set size vs threshold)
----------------------------------------------------------------------
   This plot is generated by:

       cd sort-llm
       python compare_candidateset.py

   It loads pre-computed statistics from:
     - plots/statistics_data.npz           (without layer norm)
     - plots_N128_K16_L2_H1_E32/statistics_data.npz  (with layer norm)

   These .npz files are produced by running next_num_statistics.py
   (without layer norm) and next_num_statistics_loadcheckpoint.py
   (with layer norm) respectively.

2. compare_topk_inclusion.png  (Right panel: top-k inclusion %)
---------------------------------------------------------------
   This plot is generated by:

       cd sort-llm
       python compare_topk.py

   It loads pre-computed data from:
     - plots_withoutlayernorm/topk_inclusion_data.npz
     - plots_withlayernorm/topk_inclusion_data.npz

   These .npz files are produced by running new_next_num_statistics.py
   with wlnorm set to 'without' and 'with' respectively. Each run
   evaluates 100 random batches and saves the top-k inclusion
   percentages for k = 1..10.

3. LaTeX snippet
----------------
   topk_and_candidateset.tex contains a figure environment that places
   both plots side by side with a combined caption. Include it in your
   paper with:

       \input{results/topk_and_candidateset/topk_and_candidateset.tex}

   Requires the subcaption package:  \usepackage{subcaption}
